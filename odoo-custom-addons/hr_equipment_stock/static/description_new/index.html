Equipment Maintanance Request with Stock and Purchase Management

Inventory and Purchase Requisition from Equipment Maintanance Request - Odoo Apps

This module allow your company to track employee equipment Maintanance request and procedure handle by Maintainer Location. Maintainer Manager can add list of products/items which are going to consume during Maintanance request of equipment.


Main Features:
  • Allow Maintanance Manager to track Employee Equipment Maintanance Request and using procedure handle by Maintainer Location.
  • Allow your Maintainer Team to add list of inventory items which are going to consume during Maintanance Request of Equipment.
  • System will allow you to create Internal Transfer for all Products/items which are available in your Company Stock Warehouse and If stock is not available you will be allow to create Purchase Requisition.
  • Maintainer Responsible can use to check its on Maintanance Stock location and can updates in Internal transfer if needed. This is needed for example if Maintainance Location already have some items so no need to ask to Stock warehouse.
